Computes the "hypervolumes" of the given attainment levelsets.

This computes the surface encompassed by each levelset, up to the extremum point, and returns a map between the level index and of the corresponding surfaces.

Warning
The nadir reference point is computed automatically as the extremum of all the levels. Thus, if you call this function several times on different levels, the surfaces will (may) be computed with different nadir points. To indicate your own nadir point, use the stat::Surface class directly.
